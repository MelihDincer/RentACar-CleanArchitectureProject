## RentACar Udemy Project
### Engin Demiroğ'un Udemy'de yayına almış olduğu CQRS ve Clean Architecture mimarilerini benimsediği son teknolojiler ile yazılmış olan projenin uygulanarak anlatıldığı çok kampsamlı bir eğitim setinde; araştırarak ve uygulayarak kendimi backend kısmında daha da geliştirmekteyim. Eğitimi tamamladıktan sonra dökümantasyon güncellemesi yapılacaktır. 
**Not:** Projenin Core katmanı ayrı bir projede tutulmaktadır.
